这是[omi-wc-demo](https://github.com/McCarthey/omi-wc-demo)组件库的使用文档

## 安装

需要先安装 omi，才可以使用该组件库

#### 通过 npm 安装

```
npm i omi omi-wc-demo
```

#### 使用 yarn 安装

```
yarn add omi omi-wc-demo
```

## 使用

以[header](/components/header)组件为例：

### Vue 中使用

```jsx
<template>
    <div>
        <dm-header />
    </div>
</template>

<script>
import 'omi'
import 'omi-wc-demo/build/header/'
    // ...
</script>
```

### React 中使用

```jsx
import 'omi'
import 'omi-wc-demo/build/header'

export default class Demo extends React.Component {
    render() {
        return (
            <div>
                <dm-header />
            </div>
        )
    }
}
```
