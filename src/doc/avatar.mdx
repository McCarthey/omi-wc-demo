### 使用

```jsx
import '../build/avatar'

<dm-avatar src="avatar.jpg" notices="80" />
```

其中 header 组件比就使用了 avatar 组件

### 属性

```js
static propTypes = {
    src: String,
    name: String,
    size: Number,
    color: String,
    bgColor: String,
    notices: Number
}

// 属性默认值
static defaultProps = {
    src: '',
    name: 'Admin',
    size: 40,
    color: '#fff',
    bgColor: '#87d068',
    notices: 0
}
```
